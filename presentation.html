ระบบสนับสนุนการตัดสินใจของ อสม.
เพื่อช่วยเหลือผู้ป่วยในภาวะอุทกภัย จังหวัดปทุมธานี

แพลตฟอร์มข้อมูลภูมิสารสนเทศ (GIS) สำหรับอาสาสมัครสาธารณสุขประจำหมู่บ้าน (อสม.) ใช้วางแผนเส้นทางเข้าช่วยเหลือผู้ป่วย และส่งต่อไปยังโรงพยาบาลที่ไม่ถูกน้ำท่วมหรือศูนย์อพยพ ในสถานการณ์น้ำท่วม

Phase I — Big Data (ปัจจุบัน) Phase II — Analyse Data (ถัดไป)
ที่มาและความสำคัญ

เมื่อน้ำมา… อสม. ต้องตัดสินใจเร็วและถูกต้อง

จังหวัดปทุมธานีเป็นพื้นที่ลุ่มต่ำริมแม่น้ำเจ้าพระยาที่เผชิญน้ำท่วมซ้ำซาก (เช่น มหาอุทกภัย พ.ศ. 2554) ผู้ป่วยติดเตียง ผู้ป่วยโรคเรื้อรัง และผู้สูงอายุ คือกลุ่มเปราะบางที่สุดเมื่อเส้นทางถูกตัดขาด

คำถามที่ อสม. ต้องตอบ

  • ผู้ป่วยอยู่ตรงไหน อาการรุนแรงแค่ไหน
  • ถนนเส้นไหนยังใช้รถได้ เส้นไหนต้องใช้เรือ
  • โรงพยาบาล/ศูนย์อพยพใดที่ยังปลอดภัย

ปัญหาปัจจุบัน

  • ข้อมูลกระจัดกระจาย ไม่อยู่ในระบบเดียว
  • ไม่รู้ความลึก/ความเร็วน้ำบนเส้นทาง
  • ไม่มีเครื่องมือช่วยจัดลำดับความสำคัญ

แนวทางแก้

  • รวมข้อมูล GIS ทุกชั้นไว้ในเว็บแอปเดียว
  • วิเคราะห์ด้วยแบบจำลองการไหลของน้ำ
  • แนะนำเส้นทางช่วยเหลือ-ส่งต่อ
Phase I — Big Data (กำลังดำเนินการ)

รวบรวมข้อมูลภูมิสารสนเทศเข้าสู่แพลตฟอร์มเดียว

📍 ตำแหน่ง / กายภาพ

  • โรงพยาบาล / รพ.สต. / คลินิก (86 แห่งในจังหวัด)
  • สถานีวัดระดับน้ำ (เจ้าพระยา)
  • แม่น้ำ ลุ่มน้ำหลัก (สทนช.)
  • การใช้ที่ดิน พ.ศ. 2568
  • ขอบเขตจังหวัด / อำเภอ (7) / ตำบล (60)

🌊 พื้นที่น้ำท่วมในอดีต

  • ขอบเขตน้ำท่วมรายปี พ.ศ. 2548–2564 (GISTDA)
  • Clip ให้เหลือเฉพาะเขตจังหวัดปทุมธานี
  • เลือกดูรายปี เปรียบเทียบขอบเขตได้

🧍 ข้อมูลที่จะเพิ่มต่อไป

  • ตำแหน่งผู้ป่วย + โรคประจำตัว ยา เครื่องมือแพทย์
  • สมาชิกครัวเรือน / ผู้ดูแล (อายุ เพศ)
  • Facilities: ศูนย์ชุมชน ศาลาอเนกประสงค์ จุดรวมพล
  • ความสูงพื้นที่ ขนาดถนน ประเภทบ้าน เส้นทางอพยพ

ทุกชั้นข้อมูลจัดเก็บเป็น GeoJSON (WGS84) ผ่านกระบวนการ clip ด้วยขอบเขตจังหวัด แสดงผลบนเว็บแอป Leaflet
