# ระบบสนับสนุนการตัดสินใจ อสม. — อุทกภัยจังหวัดปทุมธานี

Decision Support System for Village Health Volunteers (อสม.) in Flood Emergencies, Pathum Thani Province, Thailand

## ภาพรวม

เว็บแอปแสดงข้อมูล GIS เพื่อสนับสนุนการตัดสินใจของ อสม. ในการเข้าช่วยเหลือผู้ป่วยและส่งต่อไปยังโรงพยาบาลที่ปลอดภัยจากน้ำท่วมหรือศูนย์อพยพ

- **Phase I (Big Data)** — ปัจจุบัน: รวบรวมและแสดงข้อมูล GIS (โรงพยาบาล, พื้นที่น้ำท่วมในอดีต 2548–2564, สถานีวัดน้ำ, การใช้ที่ดิน, ขอบเขตปกครอง) ทุกชั้นข้อมูล clip เฉพาะเขตจังหวัดปทุมธานี
- **Phase II (Analyse Data)** — ถัดไป: แบบจำลองการไหลของน้ำ (Tr2–Tr100), วิเคราะห์ความลึก/ความเร็วน้ำ, พื้นที่เสี่ยงจากความเปราะบาง, การหาเส้นทางช่วยเหลือ-ส่งต่อผู้ป่วย

## การใช้งาน

- `index.html` — เว็บแอปแผนที่ (Leaflet) เปิด/ปิดชั้นข้อมูล เลือกปีน้ำท่วม สลับ OSM/ภาพดาวเทียม
- `presentation.html` — สไลด์นำเสนอโครงการ
- `data/` — GeoJSON (WGS84) clip ด้วยขอบเขตจังหวัดปทุมธานี

รันในเครื่อง: `python -m http.server` แล้วเปิด http://localhost:8000

## แหล่งข้อมูล

GISTDA (flood archive), กระทรวงสาธารณสุข (สถานพยาบาล), กรมพัฒนาที่ดิน (การใช้ที่ดิน 2568), สทนช. (ลุ่มน้ำ), กรมชลประทาน (สถานีวัดน้ำ), กรมการปกครอง (ขอบเขตปกครอง)
